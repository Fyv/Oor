# Ostéos On Rails

## Une web App pour les Ostéos

### Histoire
Développée d'abord avec Java et Swing grâce aux conseils d'une Ostéopathe, l'application migre vers le Web. Issue à l'origine d'une demande spécifique, le projet est devenue pour moi l'occasion de découvrir de nouvelles technos.

### Objectifs
Créer une alternatives gratuites et Open Sources aux outils proposés, tout en concevant une application fiable et puissante qui restera facile d'utilisation est agréable visuellement

### Outils
* Groovy avec Grails
* Twitter Bootstrap (a skinner)
